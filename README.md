## PayBills

> Fullstack web application.

**Category** - 0010

A MERN stack app to handle utility bill payments and user management.

**Visit live site:** [PayBills](https://paybills-assignment-10-0010.vercel.app/)

##### Key features

- **Bill Management**: Add, update, and delete bills with ease.
- **Secure Payments**: Safe and smooth payment process for multiple utilities.
- **Recent Bills Section**: View and track the latest bill payments.
- **User Dashboard**: Manage user information and payment history.
- **Responsive & Themed UI**: Modern design with dark/light mode support.
